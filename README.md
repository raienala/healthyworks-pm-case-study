# HealthyWorks Recreation and Wellness Intranet PM Case Study

This project is a project management case study for the HealthyWorks Recreation and Wellness Intranet Project. The project focuses on planning and managing a digital wellness platform designed to improve employee wellness participation, engagement, tracking, and reporting.

The case study includes project management deliverables such as a project charter, cost estimate, work breakdown structure, stakeholder register, RACI matrix, risk register, resource planning, earned value management, and final project documentation.

## Project Overview

HealthyWorks is a digital wellness intranet project designed to support employees through wellness tracking, gamification, digital resources, reporting, and engagement features. The project was planned using project management concepts such as scope planning, cost estimation, stakeholder management, risk management, resource planning, and schedule planning.
